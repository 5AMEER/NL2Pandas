Project Specification: AI Data Retrieval Assistant (NL2Pandas)
Objective
Build a minimalist Streamlit application that uses Gemini 1.5 Flash to convert natural language queries into executable Python (Pandas) code to retrieve data from large structured files (CSV/XLSX).
1. Technical Stack
Language: Python 3.x
LLM: Google Gemini 1.5 Flash (google-generativeai)
UI Framework: Streamlit
Data Processing: Pandas, Openpyxl
Execution Engine: Python exec() with restricted globals.
2. Core Requirements & Constraints
Single File focus: Only work with one uploaded file at a time.
Token Efficiency: Only send the DataFrame schema (column names, dtypes) and exactly 3 sample rows to the LLM.
Restricted Output: The LLM must return strictly code. The result of the logic must be assigned to a variable named ans.
Local Execution: The code generated by the AI must run locally on the full dataset (up to 100k+ lines).
3. Implementation Roadmap
Step 1: Data Observer Module
Implement a file uploader for .csv and .xlsx.
Create a function to extract metadata:
df.columns.tolist()
df.dtypes.to_dict()
df.head(3).to_dict() (The 3-line constraint).
Step 2: Security & Guardrail Module
Keyword Filtering: Create a list of FORBIDDEN_KEYWORDS (e.g., os, subprocess, open, sys, shutil, __builtins__).
Validation Function: Build a function to scan the AI-generated string for these keywords before execution.
Safe Execution: Use exec(code, {"__builtins__": {}}, local_vars) to prevent system-level access.
Step 3: Logic Generation Engine
Integrate Gemini-1.5-Flash.
System Prompt: > "You are a data retrieval expert. Given a DataFrame 'df' with these columns: {cols}, write ONLY the pandas code to answer: '{query}'. Assign final result to 'ans'. Do not import libraries. If the query is not about data retrieval, return 'SECURITY_ISSUE'."
Step 4: Streamlit UI Implementation
Query Input: Use st.chat_input.
Result Preview: Display the first 10 rows of ans if it's a DataFrame, otherwise display the value.
Export Feature: Implement st.download_button for the result using io.BytesIO.
Debugging: Add an expander to show the generated code and any execution errors.
4. Error Handling
If the AI returns SECURITY_ISSUE, display: "Security issue: Cannot execute non-retrieval logic."
If exec() fails, catch the exception and display it in a user-friendly way, suggesting a query rephrase.
